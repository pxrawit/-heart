# 🫀 Heart Disease Prediction App

แอปพลิเคชันทำนายความเสี่ยงโรคหัวใจด้วย Machine Learning (Decision Tree)

## 🚀 การใช้งาน

1. กรอกข้อมูลสุขภาพ
2. กดปุ่ม "ทำนายผล"
3. ดูผลลัพธ์และความน่าจะเป็น

## 🛠️ เทคโนโลยี

- Python 3.10+
- Streamlit
- Scikit-learn
- Plotly

## 📊 Dataset

Heart Disease UCI Dataset
